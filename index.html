鉴权配置说明

方案一：OpenAI 兼容格式的 KEY

只要是 OpenAI 兼容格式都可以。网易内部推荐使用互娱服务：AIGW 文档。外网可使用 OpenAI 或 DeepSeek 官方服务。

方案二：用户维度（网易内部）

  1. 打开 ModelSpace App 管理，新建 App，生成的 App Code 将作为 X-AIGW-APP。
  2. 给 App 添加成员。
  3. 每个成员访问 权限控制台，复制 v2 Token 内容，作为 X-Access-Token。
  4. 在插件配置中添加 Headers： 
    {"X-AIGW-APP": "_wave_code", "X-Access-Token": "your_access_token"}
  5. 将 BASE URL 配置为：https://aigw.netease.com/v1
